About This Blog

Welcome to My Blog

This is a simple blog created as a demonstration of a Go-based web application. It uses static HTML templates with HTMX for dynamic content loading from a separate API.

Technologies Used:

  • Go (Golang) for both the client and API servers
  • HTML templates for static page structure
  • HTMX for dynamic content loading
  • Tailwind CSS for styling

Feel free to explore the blog posts and enjoy the content!
